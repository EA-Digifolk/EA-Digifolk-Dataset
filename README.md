# EA-Digifolk-Dataset

Dataset of **Spanish (ES), Portuguese (PT), Italian (IT), Irish (IE), Mexican (MX), and Colombian (CO) folk tunes** in symbolic MEI (Music Encoding Initiative) format. 
* MEI files include information on sources and annotations for music analysis (key/mode), segmentation, pitch, interval, and rhythm patterns, textual topics, etc.
